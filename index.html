Liberamox

Leverage the power of Arduino with C++

Liberamox: think, type, done.

The idea of the Liberamox is to abstract the C code of arduino libraries with C++ code and a lot of fetures.

Why liberamox

Contrary to other libraries, this one is not intended to implement a new hardware but to allow the bigger number to access Arduino wonders already available.

The Arduino can be program in C++ but the way it is presented, the code is written in language C. It is possible to use the C++ to ease the development and ease the maintenance.

    • Features

    The library liberamox is written in C++ in and based on Objet Oriented Programming (OOP) so everything on Arduino is an object: From the Board, the pins or even the interruptions.

    Using this way to model objects make it possible to bring these capabilities:

    • Board: check that a pins already used is not used again, coordinate all systems, load configuration dynamically
    • Interrupt: manage their creation and their call
    • Pins: control more easily the behavior of the pins
    • Log: output to serial easily don
    • And much more
